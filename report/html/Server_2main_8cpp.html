Detailed Description

FlightLink Telemetry Server - Main Entry Point.

A multi-threaded TCP server that receives and processes aircraft telemetry data using the FlightLink binary protocol. The server calculates fuel consumption rates and maintains per-aircraft statistics.

Usage: server.exe <port> <pool_size> port - TCP port to listen on (1-65535) pool_size - Number of worker threads for connection processing
