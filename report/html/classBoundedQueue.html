Detailed Description

A thread-safe queue with bounded capacity.

This queue implements the producer-consumer pattern with blocking operations. Producers block when the queue is full, and consumers block when the queue is empty. A shutdown mechanism allows graceful termination of waiting threads.

Note
This class is designed specifically for SOCKET handles
