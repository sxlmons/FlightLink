◆ process_client()

void process_client ( SOCKET client)

Processes a single client connection from first byte to disconnection.

This function runs the main read loop for a connected client, parsing incoming binary packets according to the FlightLink protocol. It handles flight start, telemetry data, and flight end messages, routing each to the FlightSessionManager for processing.

Intended as the worker callback for ConnectionManager.

Parameters
clientThe connected client socket to process
Note
This function will block until the client disconnects or an error occurs
