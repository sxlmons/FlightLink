Detailed Description

FlightLink Telemetry Client - Simulates aircraft data transmission.

A Linux-based TCP client that reads telemetry data from CSV files and transmits it to the FlightLink server using the binary protocol format. Simulates realistic flight data transmission with configurable delays.

Usage: ./client <server_ip> server_ip - IPv4 address of the FlightLink server
