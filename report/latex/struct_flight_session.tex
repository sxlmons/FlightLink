\doxysubsection{Detailed Description}
Per-\/flight in-\/progress state (not shared between threads). 

Each worker thread maintains its own \doxylink{struct_flight_session}{Flight\+Session} instance for the client it is currently processing. This structure tracks the fuel consumption calculations for a single flight.
