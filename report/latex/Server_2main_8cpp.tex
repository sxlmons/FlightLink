\doxysubsection{Detailed Description}
Flight\+Link Telemetry Server -\/ Main Entry Point. 

A multi-\/threaded TCP server that receives and processes aircraft telemetry data using the Flight\+Link binary protocol. The server calculates fuel consumption rates and maintains per-\/aircraft statistics.

Usage:\+ server.\+exe $<$port$>$ $<$pool\+\_\+size$>$ port -\/ TCP port to listen on (1-\/65535) pool\+\_\+size -\/ Number of worker threads for connection processing
