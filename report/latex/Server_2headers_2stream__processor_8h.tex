Per-\/connection read loop handler. 

Reads binary packets from the TCP stream, parses them via direct memory mapping, and routes each message type to the appropriate handler.

Processes a single client connection from first byte to disconnection. 

This function runs the main read loop for a connected client, parsing incoming binary packets according to the Flight\+Link protocol. It handles flight start, telemetry data, and flight end messages, routing each to the Flight\+Session\+Manager for processing.

Intended as the worker callback for \doxylink{classConnectionManager}{Connection\+Manager}.
