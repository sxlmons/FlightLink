\doxysubsection{Detailed Description}
Flight\+Link Telemetry Client -\/ Simulates aircraft data transmission. 

A Linux-\/based TCP client that reads telemetry data from CSV files and transmits it to the Flight\+Link server using the binary protocol format. Simulates realistic flight data transmission with configurable delays.

Usage:\+ ./\+client $<$server\+\_\+ip$>$ server\+\_\+ip -\/ IPv4 address of the Flight\+Link server
